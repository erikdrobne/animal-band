# animal-band

Audio sampler

### Setup
#### 1. Install node
#### 2. Install npm
```bash
    $ npm install -g npm
```
#### 3. Install yarn
```bash
    $ npm install -g yarn
```
#### 4. Install gulp
```bash
    $ npm install -g gulp
```
### Run
#### 1. Navigate to project dir
#### 2. Run yarn
```bash
    $ yarn
```
#### 3. Run gulp
```bash
    $ gulp
```
